Excel occasionally appears to be too good to be true Almost anything I would ever need to accomplish manually can be done automatically with just the entry of a formula Let s talk about a tip for using Excel Having a reference column with the links each row is tracking can be useful if you re using your sheet to track social media or website metrics A URL added directly to Excel ought to function as a clickable link However if you need to hyperlink words like the title of a page or the heading of a post you re following here s how Press Shift K while highlighting the words you want to hyperlink A box will then appear where you may type the linked URL Enter the URL by pasting it into this box after copying and pasting it You may also accomplish this manually by highlighting the cell and selecting Insert Hyperlink if the key shortcut isn t functioning for any reason
