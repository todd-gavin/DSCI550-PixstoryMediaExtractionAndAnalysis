Warner Bros. has officially announced that both "The Flash" and "Aquaman and the Lost Kingdom" have been delayed from their original release date.

The Flash was originally set to release on November 4
2022, but is now set to release on June 23 2023, which is an almost 7 month delay. Aquaman and the Lost Kingdom was originally set to release on December 16 2022, but is now set to release on March 17th 2023, which is an almost 3 month delay.
