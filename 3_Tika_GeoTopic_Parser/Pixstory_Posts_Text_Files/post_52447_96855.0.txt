There are 3 type of things to do
1. you Want to do it
2. you Have to do it
3. you Need to do it.

Want - Plan it

Have - Do it / find another way

Need - Should do it anyway

Want + Need - Concentrate/ Focus in this

Have + Need - Delegate/ Schedule for Later

Want + Need + Have - This should be your Purpose of doing that Thing.
